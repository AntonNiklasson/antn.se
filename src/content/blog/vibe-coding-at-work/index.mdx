title: Vibe coding at work
date: 2026-04-29
---

I have been writing code with AI assistance at work. Sometimes I
release code without actually putting a finger on the code.

I am trying to find a good balance between utilising AI, but also
not outsourcing decisions to the models.

- Plan ahead, iterate on the plan
- Use issue tracking as the source of truth — commit context in
  comments via integrations
- Consider what the definition of success is, or perhaps the end
  goal of a work session
- Worktrees for background research
